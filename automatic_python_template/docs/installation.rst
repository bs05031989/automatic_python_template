============
Installation
============


Stable release
--------------

To install Automatic_Python_Template, run this command in your terminal:

.. code-block:: console

    $ pip install automatic_python_template

This is the preferred method to install Automatic_Python_Template, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Automatic_Python_Template can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/BS/automatic_python_template

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/BS/automatic_python_template/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/BS/automatic_python_template
.. _tarball: https://github.com/BS/automatic_python_template/tarball/master
